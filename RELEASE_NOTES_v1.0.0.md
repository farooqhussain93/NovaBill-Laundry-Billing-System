# NovaBill Laundry v1.0.0 — Final Release

## Status
Final stable release.

## Scope
NovaBill Laundry is a single PC / single user / whole-PKR desktop billing system for laundry businesses.

## Finalized Features
- Dashboard with sales, expenses, profit and outstanding summary
- Invoice creation and editing
- Customer management with required and unique phone numbers
- Payment tracking with Paid, Partial and Unpaid statuses
- Whole-PKR financial model
- PDF invoice generation and regeneration
- Expense management
- Monthly reports
- Backup and restore with safety validation
- Schema migration ledger
- Backend and frontend logic tests
- Dependency audit wrapper

## Release Notes
This version is finalized for portfolio and GitHub presentation. No further feature changes are planned for v1.0.0.

## Intended Use
This project is designed as a local-first desktop application for a single laundry shop using one computer.

## Not Included
- Multi-user accounts
- Cloud database
- SaaS subscription system
- Online deployment of the desktop app
